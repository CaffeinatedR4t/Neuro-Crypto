# Project Neuro: Crypto Intelligence Hub

Neuro is a real-time cryptocurrency monitoring and intelligence dashboard that leverages AI to provide market insights.

## Architecture Overview

### Backend (Node.js/Express)
- **Database:** MongoDB (Mongoose) for storing price snapshots and user alerts.
- **AI Integration:** Google Gemini AI (`gemini-2.5-flash`) used for generating concise market summaries and coin-specific analysis.
- **External API:** CoinGecko API for real-time market data (BTC, ETH, SOL, BNB, XRP).
- **Background Workers:** A price worker (`priceWorker.js`) polls CoinGecko every 30 seconds to update the database and trigger alerts.
- **Services:**
  - `aiService.js`: Interface for Gemini AI.
  - `coinGeckoService.js`: Wrapper for CoinGecko API.
  - `alertService.js`: Logic for price alerts.

### Frontend (React)
- **UI Framework:** Tailwind CSS for styling.
- **Data Visualization:** Recharts for historical price charts.
- **Polling:** The `LiveFeed` component polls the backend every 15 seconds for price updates.
- **Components:**
  - `AISummary.jsx`: Displays AI-generated analysis for the market or specific coins.
  - `LiveFeed.jsx`: Real-time stream of price movements.
  - `CoinCard.jsx`: Individual coin statistics and performance.
  - `PriceChart.jsx`: Historical data visualization.

### Background Processing
- **Lambda:** A `processor` function (AWS Lambda) is included for additional data processing.

## Key Features
- **Real-time Price Tracking:** Continuous monitoring of top cryptocurrencies.
- **AI Market Analysis:** Instant summaries generated by Gemini AI.
- **Historical Data:** Snapshots stored in MongoDB to provide trend analysis.
- **Live Event Feed:** Visual representation of price fluctuations as they happen.

## Technical Decisions
- **Polling vs WebSockets:** Current implementation uses HTTP polling (30s on backend, 15s on frontend) for simplicity and to stay within CoinGecko free tier limits.
- **Gemini 2.5:** Utilizes the latest Gemini models for high-quality, data-driven analysis.
- **Surgical Snapshots:** Only essential price data is stored to minimize database growth while maintaining history.

## Development Setup
- **Backend:** `npm run dev` (Nodemon)
- **Frontend:** `npm start` (Create React App)
- **Environment:** Requires `GEMINI_API_KEY`, `MONGODB_URI`.
